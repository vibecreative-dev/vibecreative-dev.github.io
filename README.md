# vibecoding.dev

Personal landing page for Nurzhan — links and a list of apps/projects.

## Live site

Once GitHub Pages is enabled, the site is served from this repo's `main` branch.

## Editing

- **Apps list** — edit the `.app-card` blocks in `index.html`. Change the emoji icon, title, description, and `href`.
- **Links** — the Telegram button points to `https://t.me/nurzhqnn`.
- **Styling** — all styles live in `styles.css`.

## Enable GitHub Pages

1. Go to the repo **Settings → Pages**.
2. Under **Build and deployment → Source**, choose **Deploy from a branch**.
3. Select branch `main` and folder `/ (root)`, then **Save**.
4. The site will be available at the URL shown on that page.
